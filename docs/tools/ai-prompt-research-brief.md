# Research Prompt Brief

- **Tool ID:** `ai-prompt-research-brief`
- **Categoria:** `ai-prompt`
- **Free:** Sì
- **Descrizione breve:** Genera un prompt di ricerca strutturato per raccogliere insight e benchmark.
- **Input:** `form`
- **Output:** `json`

## Campi di input
- `topic` (richiesto) — Argomento principale della ricerca.
- `audience` (richiesto) — Audience o stakeholder principali.
- `scope` (richiesto) — Livello di approfondimento atteso.
- `deliverable` — Formato finale desiderato.

## Output
- Prompt principale: Brief generato
- Outline/Sezioni: Struttura suggerita
- Metadata: Parametri
- Suggerimenti: Consigli

## Consigli d'uso
- Restringi la finestra temporale o il mercato geografico per risultati più precisi.
- Chiedi sempre una sezione “limitations” per evitare overconfidence nei dati.
