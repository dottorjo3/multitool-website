# AI Persona Mentor Builder

- **Tool ID:** `ai-prompt-persona-mentor`
- **Categoria:** `ai-prompt`
- **Free:** Sì
- **Descrizione breve:** Costruisci un prompt per un mentore AI che accompagna utenti in un percorso guidato.
- **Input:** `form`
- **Output:** `json`

## Campi di input
- `persona` (richiesto) — Profilo sintetico della persona che riceverà il mentoring.
- `challenge` (richiesto) — Problema o traguardo che il mentore deve affrontare.
- `format` (richiesto) — Struttura temporale del mentoring.
- `tone` — Stile comunicativo desiderato.

## Output
- Prompt principale: Prompt del mentore
- Outline/Sezioni: Struttura del percorso
- Metadata: Dettagli chiave
- Suggerimenti: Suggerimenti

## Consigli d'uso
- Specifica esempi reali o casi studio per rendere il mentoring più concreto.
- Aggiungi criteri di completamento per ogni tappa così da misurare l'impatto.
