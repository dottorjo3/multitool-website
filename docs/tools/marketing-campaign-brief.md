# Marketing Campaign Brief

- **Tool ID:** `marketing-campaign-brief`
- **Categoria:** `marketing`
- **Free:** Sì
- **Descrizione breve:** Genera un brief operativo per campagne marketing multicanale.
- **Input:** `form`
- **Output:** `json`

## Campi di input
- `product` (richiesto) — Oggetto della campagna.
- `audience` (richiesto) — Pubblico target principale.
- `goal` (richiesto) — Obiettivo core della campagna.
- `channel` — Canale principale su cui investire.

## Output
- Prompt principale: Brief operativo
- Outline/Sezioni: Sezioni del piano
- Metadata: Dati principali
- Suggerimenti: Suggerimenti

## Consigli d'uso
- Definisci il budget o le risorse disponibili per ottenere raccomandazioni più realistiche.
- Aggiungi segmentazioni secondarie (es. prospect vs clienti attivi) per personalizzare i messaggi.
