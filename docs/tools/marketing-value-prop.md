# Value Proposition Builder

- **Tool ID:** `marketing-value-prop`
- **Categoria:** `marketing`
- **Free:** Sì
- **Descrizione breve:** Crea una value proposition e messaggi di posizionamento per un'offerta.
- **Input:** `form`
- **Output:** `json`

## Campi di input
- `product` (richiesto) — Prodotto o servizio da posizionare.
- `audience` (richiesto) — Segmento principale di clientela.
- `advantage` (richiesto) — Vantaggio competitivo o beneficio principale.
- `tone` — Tono generale della comunicazione.

## Output
- Prompt principale: Value prop generata
- Outline/Sezioni: Struttura messaggi
- Metadata: Punti chiave
- Suggerimenti: Suggerimenti

## Consigli d'uso
- Integra dati numerici (percentuali, tempo risparmiato) per rinforzare la differenziazione.
- Prepara versioni alternative per segmenti secondari del pubblico.
