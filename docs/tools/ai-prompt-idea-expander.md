# Prompt Idea Expander

- **Tool ID:** `ai-prompt-idea-expander`
- **Categoria:** `ai-prompt`
- **Free:** Sì
- **Descrizione breve:** Trasforma un'idea grezza in un prompt strutturato per modelli generativi.
- **Input:** `form`
- **Output:** `json`

## Campi di input
- `idea` (richiesto) — Descrizione sintetica (almeno 20 caratteri) dell'idea originale.
- `audience` (richiesto) — Pubblico principale a cui è destinato il risultato.
- `goal` (richiesto) — Obiettivo primario del prompt generato.
- `tone` — Tono richiesto per la risposta del modello.

## Output
- Prompt principale: Prompt generato
- Outline/Sezioni: Outline consigliato
- Metadata: Parametri chiave
- Suggerimenti: Suggerimenti rapidi

## Consigli d'uso
- Aggiungi KPI o metriche per orientare la risposta verso risultati misurabili.
- Indica formati finali (slide, email, piano editoriale) per ottenere output coerenti.
