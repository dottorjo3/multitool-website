# Interview Prompt Designer

- **Tool ID:** `ai-prompt-interview-designer`
- **Categoria:** `ai-prompt`
- **Free:** Sì
- **Descrizione breve:** Genera prompt per creare interviste e domande mirate su ruoli specifici.
- **Input:** `form`
- **Output:** `json`

## Campi di input
- `role` (richiesto) — Ruolo o posizione oggetto dell’intervista.
- `scenario` (richiesto) — Contesto sintetico per orientare le domande.
- `focus` (richiesto) — Area principale su cui focalizzare le domande.
- `tone` — Stile di conduzione desiderato.

## Output
- Prompt principale: Prompt intervista
- Outline/Sezioni: Struttura consigliata
- Metadata: Parametri
- Suggerimenti: Consigli

## Consigli d'uso
- Aggiungi skill specifiche o strumenti chiave per rendere le domande più concrete.
- Inserisci criteri di valutazione numerici per confrontare i candidati.
