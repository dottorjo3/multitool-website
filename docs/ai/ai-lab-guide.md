# AI Lab — Quick Guide

1. **Scegli il modello consigliato**  
   Ogni tool mostra il modello suggerito (GPT, Claude, Llama). Puoi sostituirlo con quello del tuo stack, ma inizia sempre dal preset.

2. **Personalizza i prompt**  
   - Usa i campi “Tone”, “Audience”, “Deliverables” per ridurre i passaggi di editing.  
   - Salva i prompt efficaci nel tuo knowledge base interno.

3. **Controlla i tempi medi**  
   Le badge indicano il tempo medio di esecuzione. Se superi i 15s, prova a spezzare il task o usare un modello più leggero.

4. **Esporta i risultati**  
   Ogni tool restituisce JSON strutturati: integrali nei workflow (CRM, Notion, Zapier).

> Suggerimento: crea la pipeline “Campaign Brief → Value Prop → Email Sequence” per ottenere asset allineati in pochi minuti.
